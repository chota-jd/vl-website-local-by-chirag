## 5. Keyword Density & Best Practices

### Keyword Placement Strategy

1. **Primary Keyword in Title Tag:** Always include the main keyword in the first 60 characters
2. **Natural Keyword Integration:** Avoid keyword stuffing; maintain readability
3. **Long-Tail Keywords:** Include variations like "National Learning Management System" alongside "LMS"
4. **Location-Based Keywords:** Consider adding "India" or "National" when relevant
5. **Action-Oriented Language:** Use verbs like "Transform," "Build," "Deliver," "Engineer"

### Content Optimization Tips

- Use keywords naturally in the first 100 words of content
- Include keywords in alt text for images
- Create internal linking structure using keyword-rich anchor text
- Maintain keyword density between 1-2% for primary keywords
- Use semantic variations to avoid over-optimization
